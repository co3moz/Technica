Introduction

Bushido is highly customizable and technological HTML5 e-commerce template.

Lots of components, widgets and pre-built page variations along with great premium slider makes it perfect solution for those who want to run online retail business.

Template is built with LESS and has the modular structure so you will easily compose new pages and make necessary adjustments.

Customizing Template (Editing HTML Files)

HTML Structure (Landing Page)

HTML Code devided into sections that are represented by <section> tag. Each section is a block of content. Thus, if you want to change content of Catalog Grid of Tiles you do it in section with class="catalog-grid". Document is properly commented so you won't get lost.

Editing Menu

To edit Menu you need to find code block called "<!--Main Menu-->" in any HTML file.

Note 2 important classes "has-submenu" applied to <li> element and "submenu". This for propper work of dropdowns.

Replacing Images and Slides (Hero Slider)

Removing Parallax effect

If you do not need parallax effect on images simply remove "parallax" class from <body> tag.

Editing Content in Other Blocks

To change images, texts and other content in chosen content block you need to find section with appropriate "class" attribute.

For example, to edit content of the "Brands Carousel Widget" go to section with class="brand-carousel". If you need simply replace the images but retain content structure as it is you can do that in folder: Bushido / img / brands.

Changing Social Icons

In this template we used FontAwesome font for social icons. It is Font icons which means you can easily change them by replacing fa fa-iconName class.

Changing Look and Feel (Editing LESS Files)

Setting Up LESS Environment

Less is a CSS pre-processor, meaning that it extends the CSS language, adding features that allow variables, mixins, functions and many other techniques that allow you to make CSS that is more maintainable, themable and extendable.

For more information visit LESS official website.

For compiling LESS file into CSS we used software called "Prepros". Visit it's website at http://alphapixels.com/prepros/

LESS Files Structure

LESS File Name Elements Inside
bootstrap folder All Bootstrap framework .less file.
colors folder Colores.less file - for generating different color schemes.
font-awesome folder FontAwesome icons .less files.
lightGallery lightGallery plugin .less files.
nouislider nouislider plugin .less files.

blog.less Styles of Blog pages (post, post open, sidebar)
pages.less Secondary pages styles: about, coming soon, 404, registration, contacts, support.

widgets.less Styles of all widgets: tabs widget, media links, posts twitter widget, subscription widget, gallery widget, brands carousel, testimonials widget, breadcrumbs, pagination, sticky buttons, popovers, expandable panels, accordion widget, contact info widget, sidebar search widget, countdown widget.

Setting Up Contact Form

The new version of Bushido features Ajax Contact Form that will send emails to specified by you email address. The only thing you have to do is to go to Bushido/mailer folder find file called config.ini, open it in text editor change email addres to your own and default subject (optionally). That's it!

Please note the form will not work in local environment you have to upload your site to server.
